## 架构关键决策
- **定位:** 专业 DAW 和 AI 一键生成之间的"中间地带"
- **技术:** 原生 Web Audio API，纯前端，零成本
- **存储:** localStorage（MVP 阶段，后续迁移 IndexedDB）
- **乐器:** 无采样，纯合成器模拟（Web Audio API，文件小）
- **协作:** 每个 Sprint 完成 → 通知用户测试 → 确认后继续
